Jikan API

This project uses the Jikan API, which is an unofficial public API for MyAnimeList. Jikan provides access to anime data such as titles, images, episode counts, and ratings.

Why This API?

The Jikan API was chosen because it is free to use, does not require an API key, and provides reliable, well-structured data that is easy to integrate using JavaScript.

How It Is Used

In this project, the API is used to fetch a list of anime for the homepage and detailed information for individual anime on the detail page. JavaScript fetch requests retrieve the data, which is then dynamically displayed on the website.

API Source

Data provided by the Jikan API (https://api.jikan.moe/v4/anime), which retrieves information from MyAnimeList.
