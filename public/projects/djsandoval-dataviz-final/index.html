MLB Ballpark Home Run Factors

This project compares MLB ballparks using home run park effect, elevation, temperature, wall height, field dimensions, roof factor, and daytime factor. Home run park effect uses 100 as league average. Values above 100 are more home run friendly, while values below 100 are less home run friendly.
